## Estrategias (legales y autorizadas) para identificar la IP real

Cuando tienes permiso para auditar o pentestear un sitio protegido por Cloudflare:

1. **Buscar subdominios no protegidos**
    - Algunos subdominios pueden apuntar directamente a la IP real.
    - Herramientas: `subfinder`, `amass`, `assetfinder`.
2. **Historial de DNS**
    - Usar servicios como SecurityTrails, DNSDumpster o Censys para ver registros antiguos antes de que activaran Cloudflare.
3. **Shodan / Censys / FOFA**
    - Buscar la IP en bases de datos públicas filtrando por certificados SSL o cabeceras HTTP.
4. **Revisar configuraciones MX/FTP/SMTP**
    - Muchas veces el correo (MX) no pasa por Cloudflare y revela la IP original.
5. **Explotar filtraciones**
    - Archivos de configuración filtrados (GitHub, S3) con la IP real.
    - Respuestas de API internas que devuelvan datos de host.


# favicons

https://github.com/pielco11/fav-up

- trata de buscar el favicon de un sitio

```python
import requests
from bs4 import BeautifulSoup
from urllib.parse import urljoin
import sys

def get_favicon(url):
    try:
        # Obtener contenido HTML
        response = requests.get(url, timeout=10)
        response.raise_for_status()

        soup = BeautifulSoup(response.text, "html.parser")

        # Buscar favicon en el HTML
        icon_link = soup.find("link", rel=lambda x: x and "icon" in x.lower())

        if icon_link and icon_link.get("href"):
            favicon_url = urljoin(url, icon_link["href"])
        else:
            # Fallback a /favicon.ico
            favicon_url = urljoin(url, "/favicon.ico")

        return favicon_url

    except Exception as e:
        return f"Error: {e}"

if __name__ == "__main__":
    if len(sys.argv) != 2:
        print(f"Uso: python {sys.argv[0]} <URL>")
        sys.exit(1)

    site_url = sys.argv[1]
    favicon = get_favicon(site_url)
    print(f"Favicon URL para {site_url}: {favicon}")

```
