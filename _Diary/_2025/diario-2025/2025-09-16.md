# Neovim
NeoVim (nvim) es un editor de texto altamente configurable y extensible, basado en el popular editor Vim. Nació como un proyecto para modernizar y mejorar el código base de Vim, con el objetivo de hacerlo más eficiente, fácil de usar y más versátil.

## Instalar neovim

```
brew install neovim
nvim --version

```

# fd (find)
**Fd** es una alternativa moderna y más rápida al comando `find` de UNIX. Su objetivo principal es encontrar archivos y directorios de forma eficiente. A diferencia de `find`, tiene una sintaxis mucho más simple y una serie de características por defecto que la hacen más amigable para el usuario:
## Instalar fd 
```
brew install fd
```

# fzf

**Fzf** es un **buscador difuso** (fuzzy finder) de propósito general. Su función no es encontrar archivos por sí solo, sino tomar una lista de entradas (generalmente desde la salida de otro comando como `ls`, `find` o `fd`) y permitirte filtrarla de forma interactiva y "difusa" en tiempo real.

## instalar fzf
```
brew install fzf
```
