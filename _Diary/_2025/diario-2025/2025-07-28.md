Poner VNC en un contenedor


### Comandos docker para crear la imagen

```
alias pdbvnc="cd ~/ptd && docker build -f dockerfile-vnc -t kali-vnc ."
alias pdvnc="docker run -d -p 5901:5901 --name kali-vnc kali-vnc"


docker exec -it kali-vnc vncserver


```

### Instalar cliente vnc para mac

```
brew install --cask tigervnc-viewer
```
