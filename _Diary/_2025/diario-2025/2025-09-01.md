
## Gowitness

### Sacar capturas de pantalla
```
 gowitness scan file \
  -f uaz.edu.mx \
  --write-screenshots ./screens \
  --screenshot-format png \
  --write-db sqlite
```


### Generar el reporte

```
	gowitness report generate \
	  --db-uri sqlite://gowitness.sqlite3 \
	  --screenshot-path ./screenshots \
	  --zip-name gowitness-report.zip
```
