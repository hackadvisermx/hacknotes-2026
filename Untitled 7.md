
# Hoja de Ejercicios: Ecuaciones Lineales

**Instrucciones:** Resuelve las siguientes ecuaciones para encontrar el valor de $x$.

## Forma 1: $x + a = b$
1.  $x + 5 = 12$
2.  $x - 3 = 4$
3.  $x + 9 = 2$
## Forma 2: $ax = b$
4.  $3x = 21$
5.  $-5x = 50$
6.  $4x = 6$ (Puedes dejar la respuesta como fracción)
## Forma 3: $ax + b = c$
7.  $2x + 1 = 15$
8.  $5x - 4 = 21$
9.  $-3x + 7 = -5$
## Forma 4: $ax + b = cx + d$
10. $5x + 2 = 3x + 10$
11. $8x - 3 = 5x + 9$
12. $2x - 7 = 6x + 5$
## Forma 5: $a(bx + c) = d$
13. $4(x + 2) = 28$
14. $3(2x - 1) = 15$
15. $-2(x + 5) = -14$
## Forma 6: $ax + b + cx + d = ex + f$
16. $2x + 5 + 3x + 1 = 4x + 12$
17. $7x - 2 + x - 4 = 3x + 14$
18. $5x + 10 - 2x + 3 = x - 7$
## Forma 7: $(ax + b) / c = d$
19. $(x + 6) / 2 = 5$
20. $(3x - 1) / 4 = 2$
21. $(5x + 10) / 3 = 15$
## Sección final: Ecuaciones combinadas
22. $5(x + 1) = 2x + 17$
23. $(4x + 8) / 2 = 3(x - 1)$
24. $10x - 2(x + 3) = 5x + 9$
